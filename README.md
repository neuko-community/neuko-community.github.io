# Neuko Wiki

This is the source code for the [Neuko Wiki](https://neuko-community.github.io/neuko-wiki/).

## Project Structure

- `wiki/`: Contains the wiki content and assets.
- `.vitepress/`: VitePress configuration and theme.
- `scripts/`: Utility scripts.

## Development

1. Install dependencies:
   ```bash
   bun install
   ```

2. Run development server:
   ```bash
   bun run docs:dev
   ```

3. Build for production:
   ```bash
   bun run docs:build
   ```
